紹介

本書は、執筆時に策定中であったHTML5の仕様について初学者向けに纏めた技術誌です。 HTML5で新規に追加される要素やAPIなどの概要を簡潔に纏めた内容となっています。

発刊当時はインターネット10周年を迎える年であったため、日本国内で登場した主なWebサービスの紹介なども含まれています。 また、発刊当時における主要なウェブブラウザを、けいおん！のメンバーを交えて紹介しています。
